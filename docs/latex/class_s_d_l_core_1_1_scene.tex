\doxysubsection{Detailed Description}
\doxylink{class_s_d_l_core_1_1_scene}{Scene} class is responsible for an aggregation of game objects used in a scene It stores updatable and renderable objects in a separate containers The class should support following operations\+:
\begin{DoxyEnumerate}
\item adding of objects
\item Getting a reference to an object by Object\textquotesingle{}s name
\item Getting a reference to an object by Object\textquotesingle{}s id
\item Removing of an object from the scene by a name
\item Removing of an object from the scene by Object\textquotesingle{}s id 
\end{DoxyEnumerate}
